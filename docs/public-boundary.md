# Public Boundary

This repository is a public-safe portfolio edition of a live service-business website system.

## Public-Safe

- Source HTML, CSS, and JS for already public-facing pages
- Build and validation scripts
- Clean URL, sitemap, hreflang, and SEO architecture documentation
- High-level tracking boundary notes
- Case-study material
- Placeholder visual media that preserves expected file paths

## Excluded

- `.env*`
- `.vercel/`
- generated `public/`
- `reports/`
- `artifacts/`
- `audits/`
- `docs/research/`
- `docs/active/`
- Search Console exports
- GA4 exports
- lead reports
- growth reports
- operations exports
- active outreach strategy
- active business backlog
- private customer, job, lead, or revenue records
- real customer, property, job, or workspace photos

## Rule

Public material may explain the architecture and validation pattern. It must not expose live operational data or private business decision logs.

Visual media in this edition is intentionally synthetic placeholder media. The repository keeps the production-like asset structure for engineering review while avoiding publication of real-world job imagery.
